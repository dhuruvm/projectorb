    ## What is Project Glasswing?

    Announced **April 7, 2026**, Project Glasswing brings together 12 founding organisations and 40+ additional partners to deploy **Claude Mythos Preview** — an unreleased frontier AI model — exclusively for *defensive* cybersecurity work.

    **Core premise:** AI models have reached the point where they can surpass nearly all humans at finding and exploiting software vulnerabilities. Glasswing puts those capabilities to work for defenders *before* adversaries gain equivalent access.

    ---

    ## Claude Mythos Preview

    | Property | Detail |
    |----------|--------|
    | Type | General-purpose, unreleased frontier model by Anthropic |
    | Core capability | Surpasses all but the most skilled humans at vulnerability discovery & exploitation |
    | Findings so far | Thousands of high-severity flaws across every major OS and browser |
    | Public availability | Not released — withheld until far stronger safeguards are developed |
    | Access model | Controlled — only through Glasswing's partner programme |

    ## Model Capabilities Deployed

    - **Automated vulnerability scanning** — finds high-severity bugs in large, complex codebases
    - **Threat model builder** — maps attack surfaces and prioritises where to look
    - **Exploit development evaluation** — quantifies the model's own offensive reach so safeguards can be built against it

    ---

    ## Why This Matters

    1. **Scale of the problem** — critical software (banking, healthcare, energy, logistics) contains thousands of unfixed vulnerabilities
    2. **AI capability inflection** — models can now automate discovery at superhuman speed and breadth
    3. **Proliferation risk** — comparable models will soon exist outside organisations committed to safe deployment
    4. **The window** — defenders who act now gain a durable asymmetric advantage

    ---

    ## What Glasswing Does NOT Do

    - ❌ Does **not** release Mythos Preview publicly
    - ❌ Does **not** conduct offensive operations — all findings go to the software's own maintainers/defenders
    - ❌ Does **not** replace human security researchers — it augments them at scale
